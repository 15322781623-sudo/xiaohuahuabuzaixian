## 5. 指令流程汇总

| 指令 | 用途 | 使用位置 |
|------|------|----------|
| `role_getroleinfo` | 获取角色完整数据(heroes/artifactBooks/pearlMap/enchantMap/pet/legionResearch/lordWeapon) | refreshTeamInfo / saveCurrentLineup / applyLineup 多阶段 |
| `presetteam_getinfo` | 获取阵容预设(teamInfo/weaponId/useTeamId) | refreshTeamInfo / saveCurrentLineup / applyLineup |
| `presetteam_saveteam` | 切换/保存阵容槽位 | switchTeam / refreshTeamInfo(强制刷新) |
| `hero_gointobattle` | 武将上阵(slot) | applyLineup 阶段3 / applyHeroLevel |
| `hero_gobackbattle` | 武将下阵(slot) | applyLineup 阶段2/3 / applyHeroLevel |
| `hero_exchange` | 无损换将(heroId→targetHeroId) | applyLineup 阶段1 / confirmHeroAction |
| `hero_heroupgradelevel` | 武将升级(upgradeNum) | applyHeroLevel 循环 |
| `hero_heroupgradeorder` | 武将进阶 | applyHeroLevel 达阈值时 |
| `hero_rebirth` | 武将重生(等级归1) | applyHeroLevel 降等级时 |
| `artifact_load` | 装备鱼灵(heroId, itemId, pearlId) | applyLineup 阶段5 |
| `artifact_unload` | 卸下鱼灵(heroId) | applyLineup 阶段5 |
| `pearl_exchangeskill` | 交换鱼珠技能(pearlId1, pearlId2) | applyLineup 阶段5 |
| `pearl_replaceskill` | 替换鱼珠技能(pearlId, skillId) | applyLineup 阶段5 |
| `pearl_unloadskill` | 卸下鱼珠技能(pearlId) | applyLineup 阶段5 |
| `lordweapon_changedefaultweapon` | 切换玩具(weaponId) | applyLineup 阶段7 |
| `legion_research` | 科技升级(researchId, isMax) | syncLegionResearch |
| `legion_resetresearch` | 重置科技(type) | syncLegionResearch |
| `pet_load` | 切换宠物(slotUId) | applyLineup 阶段8 |
